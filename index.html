Shorten <Semantic> HTML Tags!

Semantic Tags

Tag Shorten Tag Description
<div id=header> <header> Signifies header of a webpage within its code
<div id=nav> <nav> Navigation bar of a webpage
<div id=main> <main> Main content of a webpage
<div id=footer> <footer> Content at bottom of a webpage
<div id=section> <section> Any given section on a webpage
